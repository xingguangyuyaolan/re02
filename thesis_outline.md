## 第三章 基于注意力机制的QMIX多无人机覆盖方法（约4500字）

### 3.1 问题建模（约600字）
- 多无人机覆盖任务形式化为Dec-POMDP
- 状态空间：各无人机位姿(8维) + 激光雷达(36维) + 覆盖进度(2维) + 局部地图(49维) + 其他智能体相对位置(6维) = 101维/智能体
- 动作空间：11种离散动作映射为4维连续控制量(vx, vy, vz, yaw_rate)
- 奖励函数设计：
  - 新覆盖奖励（+0.5/格子）
  - 覆盖完成奖励（+15.0）
  - 重复覆盖惩罚（-0.1）
  - 智能体重叠惩罚（-0.15）
  - 碰撞惩罚（-1.5）
  - 出界惩罚（-5.0）
  - 存活奖励（+0.02/步）
  - 步数惩罚（-0.005/步）
  - 任务分配奖励（+0.1）

### 3.2 网络架构总览（约400字）
- 整体框架图描述（QNetworkRNN + CrossAgentGAT + QMixNet/VDNNet/StateAttentionQMixNet）
- CTDE范式：训练时使用全局状态信息，执行时仅依赖局部观测
- 数据流：观测→GRU编码→自注意力→跨智能体注意力→Q值→混合网络→Q_total

### 3.3 智能体网络设计（约800字）
- 输入层：观测向量 + 上一步动作One-Hot + 智能体ID。输入维度 = 101 + 11 + 4 = 116
- 特征提取：全连接层(fc1) + ReLU激活
- 时序建模：单层GRU网络（hidden_dim=128），处理部分可观测性
- 观测自注意力（ObservationSelfAttention）：
  - 将GRU隐状态分割为4个token，每个token维度32
  - 4头MultiheadAttention + LayerNorm + 残差连接
  - 作用：增强智能体对观测中关键特征（目标位置、障碍物、队友坐标）的关注
- 输出层：全连接层(fc2)，输出11维Q值向量

### 3.4 跨智能体图注意力网络（约600字）
- CrossAgentGAT设计：
  - 输入：所有智能体的GRU隐状态
  - 查询/键/值线性投影（hidden_dim=128，头数=4，头维度=32）
  - 缩放点积注意力 + Softmax归一化
  - 多头输出拼接 + 输出投影 + LayerNorm + 残差连接
- 稳定化训练策略：
  - 对当前智能体使用含梯度的查询，对其他智能体使用stop-gradient
  - 避免CTDE框架下的"移动目标"不稳定问题
- 作用：显式建模智能体间的协作关系，避免覆盖区域重叠

### 3.5 混合网络设计（约800字）
- VDN混合网络：Q_tot = ΣQ_i，无参数，最简值分解
- QMIX混合网络：
  - 超网络(HyperNetwork)生成非负混合权重W1、W2
  - ELU激活保证单调性约束
  - 两阶段映射：先N个智能体Q值→隐藏层(qmix_hidden_dim=64)，再→Q_total
- 状态注意力混合网络（StateAttentionQMixNet）：
  - 在QMIX基础上增加全局状态注意力模块
  - 通过Softmax计算各智能体重要性权重（乘以N保持均值≈1）
  - 单调性保持不变（Softmax权重严格为正）
  - 作用：根据全局覆盖状况动态调整各智能体对团队总价值的贡献

### 3.6 安全防护盾机制（约500字）
- 碰撞防护：基于预测位置的配对距离检测（预测时间0.6s，危险距离0.8m，紧急距离0.45m）
- 激光雷达障碍物防护：障碍物过近时强制悬停
- 边界防护：距边界1.2m内阻止向外移动，引导转向安全方向
- 决策优先级：紧急碰撞(priority=20) > 激光雷达(30) > 边界(25) > 普通碰撞(10)
- 安全盾仅在训练阶段作为动作覆盖机制，不参与梯度计算

### 3.7 训练算法（约500字）
- TD(λ)损失：L(θ)=E[(Q_tot(s, a) - r - γ·Q_tot(s', a*))²]
- Double DQN[17]目标网络更新：每200步软更新Target网络
- ε-greedy探索策略：ε从1.0线性衰减至0.05（50,000/60,000步）
- 经验回放：EpisodeReplayBuffer，容量5000条完整轨迹
- 梯度裁剪：max_norm=10.0
- 训练间隔：每2个环境步执行1次梯度更新

### 3.8 本章小结（约300字）

### 4.2 评价指标定义（约400字）
- 覆盖率(Coverage Rate)：已覆盖格子数/总格子数（目标80%）
- 全覆盖成功率(Full Coverage Success Rate)：达到目标覆盖率的回合比例
- 碰撞率(Collision Rate)：发生碰撞的智能体比例
- 碰撞步率(Collision Step Rate)：碰撞步数/有效评估步数
- 重复覆盖率(Repeated Coverage Rate)：重复访问格子步数/总活跃步数
- 智能体重叠率(Overlap Rate)：同格子多智能体占用的步数比例
- 出界率(Out-of-Bounds Rate)：出界智能体比例
- 覆盖完成时间(Coverage Completion Time)：首次达到目标覆盖率的步数
- 累计奖励(Episode Reward)

### 4.3 课程学习训练方案（约400字）
- 三阶段课程学习设计理念：从简单到复杂，逐步增加环境复杂度
- Stage 1 → Stage 2 → Stage 3 的跨阶段权重迁移策略
  - 网络权重继承前阶段已训练参数
  - 训练计数器、epsilon重置至当前阶段初始值
  - 每阶段独立运行目录，避免指标污染

### 4.4 消融实验设计与结果分析（约1600字）

#### 4.4.1 实验组设置（约300字）
**阶段一（开放空域）消融对比：**

- 纯QMIX（无注意力机制）
- QMIX + 自注意力（Self-Attention Only）
- QMIX + 跨智能体注意力（Cross-Agent GAT Only）
- QMIX + 混合注意力（Mixing Attention Only）
- QMIX + 全注意力（三种注意力同时启用）

**阶段二（简单障碍）验证：**
- 选择阶段一最佳模型 + 全注意力模型，在障碍环境中验证策略迁移能力

#### 4.4.2 阶段一结果分析（约800字）
- 六组消融实验横向对比（用表格呈现最终指标）

  - 跨智能体注意力对降低重叠率效果显著（显式建模协作关系）
  - 全注意力在覆盖完成时间和成功率上综合最优
  - 混合注意力对覆盖完成时间有正向影响（动态调整智能体重要性）
- 各组的学习曲线对比（reward、loss、epsilon衰减）
- 任务指标对比（覆盖率、碰撞率、重复覆盖率、重叠率、出界率）
